# freq_player
Play custom frequencies

![screen](img/screen.png)

## Build
- Setup venv
`python -m venv .university`
- Install requirements 
`pip install -r requirements.txt`
- Build exe
`pyinstaller app.py --clean  --onedir`

## Run app native
- Setup venv
`python -m venv .university`
- Install requirements 
`pip install -r requirements.txt`
- Run app
`python app.py`
